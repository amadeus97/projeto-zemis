# Vuex Authentication

## Build Setup

``` bash
# instalar dependências
npm install

# iniciar servidor em localhost:8080
npm run serve
```

## Server Setup
Dentro da pasta server:
```bash
# instalar dependências
npm install

# iniciar servidor em localhost:3000
node app
```
